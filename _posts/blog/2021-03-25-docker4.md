## [인프런] 도커 명령어 


### 1. 도커 명령어 - 내부 파일 구조 보기/ 컨테이너 나열하기

- 도커 실행하기

```
docker run 이미지이름
```

- 이미지 내부 파일 시스템 구조 보기
 
```
docker run 이미지 이름 ls
```

- 실행되고 있는 컨테이너 나열하기

```
docker ps
```

- 원하는 항목 보기

```
docker ps --format 'table{{.Names}}table{{.Image}}'
```

- 모든 컨테이너 나열

```
docker ps -a
```
